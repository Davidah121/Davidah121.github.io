Untitled Project

The main title theme and what was an in progress theme for gameplay for an unfinished sci-fi game developed for the Aurora Game Development Club

The main theme for a quick 2 hour or so game jam developed for the Aurora Game Development Club.

It was a fighting game based on an existing fighting game called Footsies which is based on the term used very commonly among fighting game players.

Tetris24h

The iconic tetris theme A for the project Tetris24h. As the name implies, this was made within that 24 hour time limit.

Note that the file format is a .wav format because it is the easiest file format to read for audio.
