SMPL

A static C++ library for general purpose development.

Features include:

  • Input for Keyboard, Mouse, DInput and XInput devices
  • Math functions for a variety of transformations
  • XML and JSON document loading
  • A series of Graphics functions including a GUI system
  • An audio recording and playback system
  • Networking capabilities
  • Partial Linux support

6502 Emulator

A C++ emulator for the 6502 processor used in the NES.

Features include:

  • An Assembler and Deassembler
  • A memory viewer
  • Cycle count
  • Basic debug features allowing the user to advance one instruction at a time.
